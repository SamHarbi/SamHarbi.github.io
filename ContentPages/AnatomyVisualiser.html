Obesity Anatomy Visualiser (Javascript/HTML/CSS)

Resposive AR/3D Webapp built for medical students

I worked as part of a multidisciplinary project to create an Augmented Reality application for medical students to understand how obesity can affect the heart and lungs. My role in the project was to create the application itself while working with the medical artists on my team to incorporate the animated models created into the application.

The application itself was made with AR.js for marker based AR and Google's Model Viewer API for the 3D view, with the UI and UX using Bootstrap.
